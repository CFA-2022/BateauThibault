## Développement
#### Frotnend
Pour exécuter ce projet Angular, installez-le localement à l'aide de npm :
```
$ npm install - g @angular/cli
$ cd FrontendAngular
$ ng serve
```

#### Backend
- Si vous avez Docker : 

```
$ cd Bateauthibault
$ docker-compose up -d
```


- Si vous n'avez pas Docker : 

```
$ cd Bateauthibault
$ pip3 install -r requirements
$ python3 run/manage.py runserver
```
